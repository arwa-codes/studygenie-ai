# Student Academic Companion Chatbot

An AI-powered academic companion chatbot that helps BCA/MCA students create personalized study plans and prepare efficiently for exams through a conversational interface.

## Project Overview

The Student Academic Companion Chatbot is designed to help students organize their studies and prepare for exams more effectively. The chatbot interacts with students, collects information about their exam schedule and study preferences, and generates a personalized study plan.

This project is being developed as an MVP (Minimum Viable Product) for a single subject using HTML, CSS, Python (Flask), and SQLite.

## Features

* Chat-based academic assistant
* Interactive questionnaire
* Personalized study schedule generation
* Daily and weekly study plans
* Notes and resource management
* Motivational study tips

## Chatbot Workflow

1. Student selects a subject.
2. Chatbot asks for:

   * Exam date
   * Total syllabus/topics
   * Daily available study hours
   * Current preparation level
   * Preferred study timing
3. The backend processes the collected information.
4. A personalized study timetable is generated.

## Tech Stack

* Frontend: HTML, CSS
* Backend: Python (Flask)
* Database: SQLite
* Version Control: Git & GitHub
